Connected iPhone-to-Mac exports are now more dependable, especially for large and multi-year archives.

Improved
• Saved manual IP connections can securely reconnect after the first pairing without requiring a new code each time.
• Large connected exports use durable, resumable transfers and incremental file writing to reduce memory pressure.
• Daily Notes Only and custom schedules carry through connected Mac exports more consistently.
• Failed and interrupted jobs preserve their intended dates and provide clearer recovery details.
